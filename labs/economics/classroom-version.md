# Version Aula

## Duracion sugerida

45 a 60 minutos

## Objetivo

Introducir la metodologia en formato de clase sin cubrir todas las herramientas con la misma profundidad.

## Recorte recomendado

- Scientific Question
- Scientific Language
- Scientific Corpus
- Scientific Evidence
- Scientific Memory
- Scientific Communication

## Resultado esperado

El grupo entiende la logica del Research Intelligence Workflow y puede continuar el resto como tarea.
