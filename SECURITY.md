# Security Policy

## Scope

Research AI Lab is primarily a documentation and methodology repository, but we still treat security and integrity seriously.

This policy applies to:

- Repository content integrity
- Scripts or templates added in the future
- Prompt libraries and workflow examples
- Any future software or agent components

## Reporting

If you discover a security issue, please report it privately to the maintainers before public disclosure.

Include:

- A clear description of the issue
- Steps to reproduce, if relevant
- Potential impact
- Suggested mitigations, if known

## Principles

- Protect contributors and users
- Prefer responsible disclosure
- Document fixes clearly
- Preserve trust in research workflows and artifacts

## Future Expansion

As the project grows into agents, tooling, or hosted services, this policy should be expanded with:

- Supported versions
- Disclosure timelines
- Severity classification
- Response expectations
