# RAL-005: Roadmap

## Version 1.0

### Objective

Establish the full foundation of Research AI Lab as a credible open initiative.

### Expected Deliverables

- Core governance and identity documents
- Repository architecture
- Documentation standards
- Toolkit specification
- Economics Lab
- Public-facing README and contribution framework

### Release Criteria

- Internal consistency across all foundational documents
- Clear architecture for long-term expansion
- One lab that demonstrates the full method

## Version 1.1: Education Lab

### Objective

Extend the methodology into education-focused research questions.

### Expected Deliverables

- Education Lab specification and canonical question
- Education-specific examples and references
- Adapted toolkit assets where needed

### Release Criteria

- Education Lab conforms to RAL lab specification
- Reusable educational workflows documented

## Version 1.2: Artificial Intelligence Lab

### Objective

Apply the framework to research on AI itself.

### Expected Deliverables

- AI Lab specification
- Model evaluation and benchmark guidance
- Expanded evidence review patterns for AI claims

### Release Criteria

- AI-specific methodological risks are explicitly addressed
- Documentation remains vendor neutral and evidence driven

## Version 2.0: Research Agents

### Objective

Introduce bounded, auditable agentic workflows aligned with the documented methodology.

### Expected Deliverables

- Agent role definitions
- Workflow guardrails
- Human review protocols
- Traceability requirements

### Release Criteria

- Agents operate inside explicit methodological boundaries
- Human accountability remains central

## Version 3.0: Research Operating System

### Objective

Create a unified environment where labs, toolkit assets, resources, and agents operate as an integrated system.

### Expected Deliverables

- Shared workflow architecture
- Multi-lab orchestration patterns
- Scalable contribution model

### Release Criteria

- Growth does not compromise coherence
- The system remains teachable, extensible, and reviewable
